Contributions are welcome through pull requests on the [GitHub repository](https://github.com/luigifcruz/stelline).

## Guidelines

- Stelline follows the CyberEther code style and build patterns, so the upstream [contributing](https://cyberether.org/docs/contributing) guidelines apply here as well.
- When adding or changing a block, keep the documentation embedded in its header and the matching page in this documentation in sync.
- All contributions are considered to be licensed under the terms of the [MIT License](https://github.com/luigifcruz/stelline/blob/main/LICENSE).
